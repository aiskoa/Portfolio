**Before starting.**

Any tool shown here must be used with responsibility and extreme care, the malicious use of any of them is the responsibility of the user.

___

[Shodan](https://www.shodan.io)

Shodan is a search engine for the internet of things, it can search by IP and explore multiple devices on the network.

- Ideal for server analysis
- Offers filters for more specific searches
- Use for security audits
- Information gathering
- Physical location of the server

[Hunter](https://hunter.io)

Hunter is a professional email search engine for workers, by entering an organization's domain it will return a list of emails associated with the domain and the source from which it was obtained.

[Emailrep](https://emailrep.io)

Emailrep analyzes the entered email, provides information about the email such as its reputation and associated accounts, such as social networks among others.

___

[Truecaller](https://www.truecaller.com)

Truecaller allows obtaining information from phone numbers.

- Allows social engineering attacks
- Allows SMS Gateway attacks (Only in some cases)

*CLARIFICATION: I DO NOT RECOMMEND USING TRUECALLER AS A FIRST METHOD, AS TRUECALLER STORES YOUR LOCAL INFORMATION SUCH AS PHONE NUMBERS, NAMES AND ADDRESSES TO INCREASE ITS DATABASE AND THUS SHARE IT WITH THE WORLD.*

**Data monitoring and analysis.**

[Chatwatch](https://chatwatch.net/)

It allows monitoring a WhatsApp number or multiple of them, if the line is colored in a dark purple it means that it is online, this serves to see if two people are interacting, it requires a paid account but its demo version allows its use with limitations.
